Python utility to render pictures of handwriting as LaTeX documents.

## Example calls

Render two images:

`python texit.py tests/batch1/IMG_9562.png tests/batch1/IMG_9563.png`

Render two images, and pipe the output to a tex file:

`python texit.py tests/batch2/IMG_9562.png tests/batch2/IMG_9563.png > pre-edits.tex`
